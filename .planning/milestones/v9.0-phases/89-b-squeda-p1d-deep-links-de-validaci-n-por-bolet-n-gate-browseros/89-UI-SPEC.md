## Color

60/30/10 split is LOCKED at the design-system level (DESIGN-SYSTEM §1.1). Values are HSL tokens in `globals.css` — **zero hex in code** (constraint). Below the hex is the informational equivalent only.

| Role | Token (value) | Usage |
|------|---------------|-------|
| Dominant (60%) | `--background` = `hsl(40 33% 97%)` ≈ #F9F6F0 crema | Page/section surface — the validation section sits on `--background` (no petroleum fill) |
| Secondary (30%) | `--card` = `hsl(40 30% 99%)` ≈ #FDFBF7 / `--muted` = `hsl(40 20% 93%)` | The bordered validation block surface (`bg-card` or `bg-muted`) + `--border` = `hsl(40 16% 86%)` for separators |
| Accent (10%) | `--accent-product` = `hsl(183 38% 26%)` ≈ #2A5859 petróleo | **Reserved for the deep-link affordances only** (see below) |
| Destructive | `--destructive` = `hsl(0 72% 42%)` | Not used — this phase has **zero destructive actions** |

Accent (petróleo `--accent-product`) reserved for:
- The deep-link anchor text + its underline (link-affine treatment, matching `.net-b-link` / `.net-prov__enlace`).
- Focus-visible outline on the deep-links (`outline: 2px solid hsl(var(--accent-product))`, matching `.net-b-row:focus-visible`).

Petróleo is **never** a fill/background here, **never** applied to the "según fuente al {fecha}" caption or the R2 respaldo legend (those are `--muted-foreground`), and **never** used to signal source identity (Senado vs Cámara carry a text label, not a color — anti-insinuación LOCKED, consistent with `.net-chip` rule).

Provenance badge tokens (`--provenance-bg/-border/-fg`) are consumed as-is via `ProvenanceBadge`; `esStale` amber (`text-amber-700 border-amber-400`) is the badge's own honest-freshness signal, not a new color.

## Copywriting Contract

Anti-insinuación LOCKED: copy is factual, never causal, never evaluative. The linter bans `score`/`ranking`/`nivel`/etc. — none appear below. Trazabilidad is the rector principle: source + date + link, never intención ni causalidad.

| Element | Copy |
|---------|------|
| Section heading (h2) | **Valida este dato en la fuente** |
| Senado link label | **Ver en el Senado ↗** (secondary text: "Ficha de tramitación oficial") — rendered SIEMPRE (boletín en mano) |
| Cámara link label | **Ver en la Cámara ↗** (secondary text: "Ficha de tramitación oficial") — rendered SOLO si `prm_id_camara != null` |
| BCN | **Omitido del DOM** — sin `idNorma` no se muestra fila, ni placeholder, ni "no disponible", ni link genérico de búsqueda (fail-honest LOCKED) |
| Capture date | **según fuente al {fecha_captura}** (via `ProvenanceBadge` "Actualizado {relativeTimeEs} · {fuente}"; the "según fuente al" framing is the section's inline provenance line) |
| Respaldo snapshot R2 | **Respaldo del {fetched_at} · hash {content_hash.slice(0,12)}…** — SIN link de descarga; leyenda debajo: **"Esto decía la fuente ese día."** |
| Empty state (proyecto sin ningún link ni respaldo) | Senado siempre existe → la sección nunca queda vacía. Si por algún motivo faltara el boletín, la sección se omite del DOM (no heading huérfano) |
| Error state | Un fallo real de DB/red se **lanza** (patrón #34: nunca degradar un error a "sin dato"). La página de error honesta de la ruta lo maneja; esta sección no fabrica un empty a partir de un fallo |
| Destructive confirmation | **N/A — no hay acciones destructivas en esta fase** |

Interaction contract for external links (LOCKED constraint):
- `target="_blank"` + `rel="noopener noreferrer"` on every deep-link.
- Visible external indicator: trailing `↗` glyph (matches `ProvenanceBadge` "fuente oficial ↗").
- `aria-label` naming the source + "(abre en nueva pestaña)" (matches `ProvenanceBadge` pattern).
- Every href passes `safeExternalHref` (rejects `javascript:`/`data:`); boletín and prmID go through `encodeURIComponent`; hosts are fixed (`tramitacion.senado.cl` / `www.camara.cl`) — never built from free input.
- Clear source label per link (Senado / Cámara) — LOCKED.

Fail-honest matrix (per-datum, not per-flag — there is NO feature flag for this section, RESEARCH §Deploy):

| Datum | Present | Absent |
|-------|---------|--------|
| Senado (boletín) | "Ver en el Senado ↗" | never absent |
| Cámara (`prm_id_camara`) | "Ver en la Cámara ↗" | row omitted (no link) |
| BCN (`idNorma`) | — (not implemented this phase) | row omitted (no placeholder) |
| Respaldo R2 (`source_snapshot` prefijo `tramitacion/*`) | fecha + hash abreviado + leyenda | line omitted (no fabricated backup) |

Responsive: mobile 390px must render every link at ≥44px touch height, source label + `↗` on one legible row, no horizontal overflow (mono boletín may wrap).
