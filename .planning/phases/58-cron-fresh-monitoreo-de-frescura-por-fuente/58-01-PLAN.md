<objective>
Build the `@obs/freshness` package: a lightweight operator CLI that answers "is each ingest source
fresh?" with a single command, zero writes, zero DDL, zero network beyond read-only SELECTs.

Purpose: CRON-05 — the operator must detect staleness without digging through logs. This CLI
delivers the three signals per source (last GH Actions run, last R2 snapshot, last Supabase upsert)
plus a stale/ok flag against configured per-source thresholds.

Output:
- `packages/freshness/` — new package with catalog, query-runner, evaluate, CLI
- `package.json` root script `freshness`
- `docs/runbooks/cron-local-fallback.md` section 4
</objective>
