Signatura: formatNombre(raw: string | null | undefined): string

Reglas LOCKED:
1. null/undefined/whitespace-only -> "" (los callers conservan su null-fallback existente).
2. Passthrough guard (load-bearing): si el string contiene CUALQUIER mayúscula Unicode -> devolver VERBATIM. Usar /\p{Lu}/u (NO /[A-Z]/): fila real "fundación mas familia Ñuble" tiene Ñ y CERO A-Z; un guard ASCII la re-casearía mal.
3. Solo transforma strings 100% minúsculas: colapsar runs de whitespace a 1 espacio, split por espacio.
4. PARTICULAS (Set, LOCKED): de, del, la, las, los, van, von, y. Partícula queda minúscula EXCEPTO si es el primer token (primer token SIEMPRE capitalizado).
5. Tokens no-partícula: capitalizar la 1ª letra de cada SUB-token separado por - y ' (delimitadores preservados): "o'higgins" -> "O'Higgins", "perez-mackenna" -> "Perez-Mackenna".
6. NUNCA agregar tildes ("gonzalez" -> "Gonzalez"), NUNCA reordenar tokens, NUNCA normalizar puntuación interior.
