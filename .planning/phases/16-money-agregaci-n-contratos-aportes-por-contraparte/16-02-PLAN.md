<objective>
Ship the gated `/contraparte/[id]` route (MONEY-05, UI half) that renders a persona-jurídica contraparte's aggregated MONEY facts: contratos (ChileCompra) and aportes (SERVEL) in SEPARATE `mt-12` lanes, each row provenance-badged, with honest 3-states, zero vote data, and zero insinuation. Output: route + not-found + two lane components + their RTL tests + types/regex additions. UI-only; no DB, no connector.
</objective>

    NEVER returns a persona-natural name, donante_id, or any RUT-as-name.
  (Optional) RPC `contrapartes_listado()` — jurídica-only listing with prefixed ids. The listing PAGE is DEFERRED.
